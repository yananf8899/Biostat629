# Project 2 webpage

This `docs/` folder is a static website served by GitHub Pages at
<https://yananf8899.github.io/Biostat629/>.

## Local preview

```bash
cd docs
python3 -m http.server 8000
# open http://localhost:8000
```

## Publishing on GitHub Pages

In the GitHub repo: **Settings → Pages → Build and deployment**, set
**Source = Deploy from a branch**, **Branch = main**, **Folder = /docs**, Save.

## Files

- `index.html` — the page content
- `style.css` — colours, fonts, layout (single source of truth)
- `figures/fig{1..5}.png` — generated by `project2_analysis.R`
- `report.pdf` — compiled from `project2_report.tex`

## Updating figures

The figures are copied from the project root after running the analysis:

```bash
Rscript project2_analysis.R
cp fig1_cea_trajectories.png    docs/figures/fig1.png
cp fig2_cea_slope_boxplot.png   docs/figures/fig2.png
cp fig3_roc_curves.png          docs/figures/fig3.png
cp fig4_calibration.png         docs/figures/fig4.png
cp fig5_conformal_singletons.png docs/figures/fig5.png
pdflatex project2_report.tex
cp project2_report.pdf docs/report.pdf
```
